Чередование сфер по разворотам: обложка — малахит + лаванда → стр. 2 малахит → стр. 3 лаванда → стр. 4 небо → стр. 5 малахит + лаванда.

04
Сферы (фон)

Дуга — а не круг

  • Класс .ds-sphere + модификатор цвета (malachite / lavender / sky).
  • Абсолютное позиционирование, глубоко за край листа — минимум −30 мм в любую сторону. Видна должна быть только дуга.
  • Размер 60–145 мм. На обложке самые крупные, на внутренних — компактнее.
  • Сферы не должны пересекаться с типографикой. Если задевают заголовок или абзац — отодвигай дальше за край.
  • Между листами не перетекают — overflow: hidden на .page.
05
Структура подборки

5 страниц на 3 канала + бонус

1. Обложка 2. Канал 01 / 03 3. Канал 02 / 03 4. Канал 03 / 03 5. Бонус (сообщество, ресурс) — всегда последняя

Если бонуса нет — 4 страницы. Канон не нарушается, просто пятая отсекается.

06
Анатомия обложки

Один заголовок, один лид, одна капсула

┌────────────────────────────────────────┐ │ [мета справа] │ ← Подборка · 0X · дата │ │ │ [eyebrow] │ │ [Title в 2 строки] │ │ │ │ [Лид 1–2 строки, max-width 70%] │ │ [Капсула-ссылка на канал автора] │ ← опционально │ │ │ [страниц·мин] │ └────────────────────────────────────────┘

Делать

  • Капсула со ссылкой — в одном месте
  • Дата выпуска — мономерным сверху справа
  • Лид — короткий, 1–2 строки

Не делать

  • Дублировать «3 канала · 1 сообщество»
  • Ставить ссылку и сверху, и снизу
  • Раздувать обложку до пяти блоков
07
Анатомия страницы канала

Без больших нумераторов

┌────────────────────────────────────────┐ │ Подборка · название 0X / 0N │ ← page-head │ │ │ [eyebrow: Канал 0X / 0N] │ │ [ИМЯ КАНАЛА — 44pt] │ │ [ссылка-моно с подчёркиванием] │ │ │ │ ┌─[shot 16:9]────┐ ┌─ratings──────┐ │ │ │ скриншот │ │ Понятность │ │ │ │ │ │ ●●●●● ●○ │ │ │ └────────────────┘ └──────────────┘ │ │ │ │ [Лид 12pt medium] [тело 11pt regular] │ │ │ │ Канал 0X / 0N [короткий тег] │ ← page-foot └────────────────────────────────────────┘
  • Никаких больших нумераторов «01/02/03» на странице — счётчик уже есть в шапке и эйбро.
  • Eyebrow: только «Канал 0X / 0N». Платформу (YouTube/Telegram) не пишем — она ясна из ссылки.
  • Маркер уровня типа «узкая тема» допустим — добавляется через микро-черту.
  • Под скриншотом — никаких подписей-плейсхолдеров.
08
Шкала рейтингов

5 точек на категорию — механика, а не список

Понятность
Универсальность
Качество материала
Бесплатность
  • Состав категорий — гибкий. На примере выше: «Понятность · Универсальность · Качество · Бесплатность». Для другой подборки набор может быть другим (например: «Глубина · Регулярность · Подача · Польза»). Фиксируется метрика категории — то, что её можно оценить от 0 до 5.
  • Количество категорий: от 3 до 5. Меньше — теряется смысл, больше — рябит.
  • Шкала всегда из 5 точек. Заполненные — 9px background: --color-ink; пустые — 7px с бордером --color-ink.
  • Структура строки: лейбл UPPERCASE Inter Medium 8.5pt сверху, ряд точек снизу, между категориями — линия --border-rule. Последняя — без линии.
  • Расщепление шкалы («если в теме / если нет») — два ряда точек с подписями допустимы для одной категории.
  • «Выход за шкалу» (7 точек + курсивная сноска .rating-note) — художественный приём, не норма.

09
Ссылки

Все ссылки кликабельны. Без исключений.

Стандартная ссылка

.entry-link: Courier Prime 10pt, серое подчёркивание --color-rule, цвет --color-ink-soft. При hover — темнеет до --color-ink. Чистый URL без суффиксов.

Капсула-ссылка обложки

.cover-channel: бледная плашка с радиусом 999px, красная точка слева, стрелка → справа. Hover — инверсия в чернильный фон. Единственная в подборке.

10
Скриншоты

Реальные превью, не плейсхолдеры

  • Источник — реальные превью видео или страницы сообщества. Не нейросетевые иллюстрации.
  • Соотношение 16:9 для каналов (для сообщества тоже допустимо).
  • Файлы лежат в assets/<slug>-shot.png.
  • Подключение: <image-slot id="shot-<slug>" shape="rect" src="assets/…">.
11–13
Шапка, подвал, цитата, экспорт

Сервисные правила

  • page-head: слева — название подборки, справа — счётчик «0X / 0N». Линия снизу.
  • page-foot: слева — позиция в подборке, справа — короткий человеческий тег. Линия сверху.
  • Цитата создателя — компонент .entry-quote: малахитовая подложка, левая полоса 3px. Только на бонусной странице.
  • Экспорт в PDF: через *-print.html с авто-печатью. В диалоге печати: Save as PDF, Margins → None, Background graphics → ON, Headers and footers → OFF.
